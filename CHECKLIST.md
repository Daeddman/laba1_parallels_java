## 📞 Що робити, якщо...

### Програма не компілюється
- Перевірте версію Java: `java -version`
- Переконайтесь, що в директорії `src`
- Подивіться помилки компіляції

### Програма не запускається
- Перевірте, чи є файли .class
- Запускайте з правильної директорії
- Перевірте права доступу

### Не розумію результати
- Прочитайте EXAMPLES.md
- Подивіться на формулу суми
- Запустіть з простими параметрами (2 потоки, крок 1)

### Git не працює
- Прочитайте GIT_GUIDE.md
- Перевірте, чи встановлений Git: `git --version`
- Використовуйте GitHub Desktop як альтернативу
