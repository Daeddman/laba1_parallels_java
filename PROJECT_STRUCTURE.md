## 📁 Опис файлів

### Код програми

#### `src/Main.java` (112 рядків)
**Головний файл програми з трьома класами:**

1. **SumThread** (extends Thread)
   - Обчислює суму: 0 + step + 2×step + 3×step + ...
   - Використовує `volatile boolean running` для зупинки
   - Виводить результати після завершення

2. **ControllerThread** (extends Thread)
   - Керує зупинкою робочих потоків
   - Зупиняє кожен потік через задану затримку
   - Виводить повідомлення про зупинку

3. **Main**
   - Точка входу програми
   - Зчитує параметри від користувача
   - Створює та запускає потоки
   - Очікує їх завершення

**Ключові особливості:**
- ✅ Безпечна зупинка потоків
- ✅ Використання volatile
- ✅ Синхронізація через join()
- ✅ Обробка InterruptedException
