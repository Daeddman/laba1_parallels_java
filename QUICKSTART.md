# ⚡ Quick Start Guide

## Швидкий старт за 3 кроки

### 1️⃣ Компіляція
```bash
cd src
javac Main.java
```

### 2️⃣ Запуск
```bash
java Main
```

### 3️⃣ Введення параметрів
```
Введіть кількість потоків: 3

Потік №1:
  Введіть крок послідовності: 5
  Введіть час роботи (мс): 1000

Потік №2:
  Введіть крок послідовності: 10
  Введіть час роботи (мс): 1500

Потік №3:
  Введіть крок послідовності: 3
  Введіть час роботи (мс): 2000
```

## 📋 Приклади запуску

### Приклад 1: Швидкий тест (2 потоки)
```bash
cd src
echo -e "2\n5\n500\n10\n1000" | java Main
```
Потік 1: крок=5, час=500мс | Потік 2: крок=10, час=1000мс

### Приклад 2: Середній тест (3 потоки)
```bash
cd src
echo -e "3\n3\n800\n7\n1200\n5\n1600" | java Main
```
Потік 1: крок=3, час=800мс | Потік 2: крок=7, час=1200мс | Потік 3: крок=5, час=1600мс

### Приклад 3: Інтенсивний тест (4 потоки)
```bash
cd src
echo -e "4\n2\n500\n4\n1000\n6\n1500\n8\n2000" | java Main
```
Потік 1: крок=2, час=500мс | Потік 2: крок=4, час=1000мс | Потік 3: крок=6, час=1500мс | Потік 4: крок=8, час=2000мс

## ⚙️ Параметри

| Параметр | Опис | Рекомендовані значення |
|----------|------|------------------------|
| Кількість потоків | Скільки паралельних потоків створити | 2-8 |
| Крок послідовності (для кожного потоку) | Приріст між елементами (0, крок, 2×крок, ...) | 1-10 |
| Час роботи (для кожного потоку) | Час роботи потоку до зупинки (мс) | 500-2000 |

**Увага:** Кожен потік має свої власні параметри (крок і час роботи)!
