## Опис
Програма демонструє використання багатопоточності в Java для обчислення сум числових послідовностей.

## Функціональність
- **Робочі потоки**: Кожен потік обчислює суму послідовності, що починається з 0 і зростає із заданим кроком
- **Керуючий потік**: Окремий потік, який зупиняє робочі потоки через заданий інтервал часу
- **Виведення результатів**: Кожен потік виводить свій номер, обчислену суму та кількість доданків

## Класи та структура

### SumThread (extends Thread)
- Обчислює суму послідовності: 0, step, 2*step, 3*step, ...
- Використовує volatile boolean для безпечної зупинки
- Виводить результати після завершення роботи

### ControllerThread (extends Thread)
- Зупиняє кожен робочий потік через заданий інтервал часу
- Виводить повідомлення про зупинку кожного потоку

## Параметри програми
При запуску програма запитує:
1. **Кількість потоків** - скільки потоків буде створено для обчислень

Для кожного потоку окремо:
2. **Крок послідовності** - значення кроку для числової послідовності цього потоку
3. **Час роботи** - час в мілісекундах, протягом якого працюватиме цей потік

## Особливості реалізації

### Валідація введення
Програма перевіряє коректність введених даних:
- **Кількість потоків**: від 1 до 32 (MAX_THREADS)
- **Крок послідовності**: тільки додатні цілі числа (>0)
- **Час роботи**: тільки додатні цілі числа в мілісекундах (>0)
- **Захист від помилок**: обробка некоректного введення (текст, від'ємні числа, нуль)

### Використання volatile
Змінна `running` оголошена як `volatile`, що гарантує правильну видимість змін між потоками.

### Безпечна зупинка потоків
Замість deprecated методу `stop()` використовується прапорець `running`, що дозволяє потоку коректно завершити роботу.

### Синхронізація
Метод `join()` використовується для очікування завершення всіх потоків перед виходом з програми.
